◆ obtenir_question_suivante()

Question obtenir_question_suivante ( void )

Obtenir la prochaine question non posée.

Cette fonction sélectionne une question aléatoire parmi celles qui n'ont pas encore été posées. Si toutes les questions ont été posées, elle retourne une question vide.

Returns
La prochaine question ou une question vide si toutes les questions ont été posées.
