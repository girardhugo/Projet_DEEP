◆ gerer_boutons()

void gerer_boutons ( Question q)

Gérer les boutons pour interagir avec les trappes.

Cette fonction gère les actions des boutons pour :

  • Naviguer entre les trappes (boutons gauche et droit).
  • Ajouter de l'argent à une trappe (bouton haut).
  • Retirer de l'argent d'une trappe (bouton bas).
Parameters
qLa question actuelle, utilisée pour mettre à jour les réponses affichées.

< Dernier temps où le bouton haut a été traité.

< Dernier temps où le bouton bas a été traité.

< État du bouton droit.

< État du bouton gauche.

< État du bouton haut.

< État du bouton bas.
