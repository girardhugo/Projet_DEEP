void machine_etats ( )

Machine à états du jeu.

Gère les différents états du jeu : DEBUT, REGLES, JEU et FIN. Chaque état correspond à une étape spécifique du jeu.

< Réinitialiser le compteur de questions.

< Réinitialiser le numéro de la question.

< Afficher l'écran de début.

< Passer à l'état REGLES.

< Afficher l'écran des règles.

< Attendre 6 secondes.

< Passer à l'état JEU.

< Si aucune question n'est disponible.

< Afficher la question et le numéro.

< Gérer les boutons pour placer ou retirer de l'argent.

< Incrémenter le compteur de questions.

< Incrémenter le numéro de la question.

< Vérifier si 10 questions ont été posées.

< Afficher l'écran "PERDU".

< Attendre 5 secondes avant de retourner à l'écran DEBUT.

< Retourner à l'état DEBUT.

< Afficher l'écran de fin classique.

< Attendre 5 secondes avant de retourner à l'écran DEBUT.

< Retourner à l'état DEBUT.
