Obtenir la prochaine question non posée. 

Cette fonction sélectionne une question aléatoire parmi celles qui n\textquotesingle{}ont pas encore été posées. Si toutes les questions ont été posées, elle retourne une question vide.

\begin{DoxyReturn}{Returns}
La prochaine question ou une question vide si toutes les questions ont été posées. 
\end{DoxyReturn}
